# Vue 到单 HTML 转换工具

这个工具用于将 Vue 项目中的页面组件转换为独立的单 HTML 文件，便于 AI 编程时的增删改操作。

## 功能特性

- 将 Vue 组件转换为独立的 HTML 文件
- 自动添加 DaisyUI 的 bordered 类名（如 input-bordered、select-bordered 等）

## 使用方法

### 安装依赖

```bash
npm install @vue/compiler-sfc
```

### 运行转换脚本

```bash
node scripts/advanced-convert-vue-to-single-html.js
```

### 输出位置

转换后的 HTML 文件将保存在 `单HTML改造/输出/` 目录下，保持原有的目录结构。

## 转换规则

### 模板转换

- `v-model` → `x-model`
- `v-if/v-show` → `x-show`
- `v-for` → `x-for`
- `@事件` → `@事件`
- `:属性` → `x-bind:属性`
- 移除`:key`属性
- `<router-link>` → `<a>`

### DaisyUI 类名添加

- `<input>` → 添加 `input-bordered` 类
- `<select>` → 添加 `select-bordered` 类
- `<textarea>` → 添加 `textarea-bordered` 类
- `<button>` → 添加 `btn-bordered` 类
- `<input type="checkbox">` → 添加 `checkbox-bordered` 类
- `<input type="file">` → 添加 `file-input-bordered` 类

### 脚本处理

- 移除 Vue 特定的 import 语句
- 转换 Composition API 为原生 JavaScript
- 使用 Alpine.js 进行数据绑定

## 输出示例

输出的 HTML 文件包含以下结构：

- 完整的 HTML 文档结构
- Tailwind CSS 和 DaisyUI 的 CDN 链接
- 原有的样式内容
- 转换后的模板内容
- Alpine.js 脚本支持
